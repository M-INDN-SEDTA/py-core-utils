## 📌 What is this?
**`py-core-utils`** is a central repository containing my core standalone Python automation utilities and algorithms. Instead of cluttering my profile with separate 1-file scripts, I consolidated them here into a clean workspace.

---

## 📦 What It Contains

This repository houses three main tools, built purely with Python's standard library:

### 1. 🌲 TreeXplorer
* **What it does:** Recursively scans a target path on your computer and prints out a clean, visual folder tree diagram directly into the terminal window.
* **Why it's useful:** Helps developers instantly map out new or massive project code bases without manual searching.

### 2. ⚡ Sortify
* **What it does:** An algorithmic engine containing custom execution benchmarks for sorting arrays and data records. 
* **Why it's useful:** Demonstrates optimized memory handling and core computer science sorting logic.

### 3. 📂 File Hierarchy Mapper
* **What it does:** Flattens complex, deeply nested computer directories and organizes them into readable, structured system data.
* **Why it's useful:** Automates folder cleanup and file scanning tasks.

## ⚙️ How to Run It

1. Clone this repository:
```bash
git clone [https://github.com/YOUR-USERNAME/py-core-utils.git](https://github.com/YOUR-USERNAME/py-core-utils.git)
cd py-core-utils

```

2. Run any tool instantly from your terminal:

```bash
python tree_xplorer.py
python sortify.py
python file_hierarchy.py

```
